Narrative and voice I

This was the basis for my narrative and voice project. At first I thought about using a moon API. But the moon as an astronomical object moves in enormous distances and timeframes. This means that the code had to run for ages for me to see any interesting movement. I found this database of some 'Lunar NIR Spectrophotometry' readings. The code runs throught the text files containing the data, stored in an external file ( had to use ajax to access the files ), and use the value it finds to create a drawing. Each value is adapted to change the radius and the color of each circle. When the code reaches the end of the file, it adds x to the y position of the circle, the x position returns to 0 and does the same with a new file.
